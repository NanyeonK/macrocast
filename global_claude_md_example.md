# Global Preferences (Chan)

## Coding

- Econometrics: R preferred. ML tasks: Python preferred.
- Python: prefer .ipynb workflow conceptually, but for package development use .py modules.
- Write code that is clearly understandable with intermediate comments explaining logic.
- Large-scale data research: track data dimensions, missing values, and edge cases explicitly.

## Writing (Academic)

- No m-dashes, colons, or semicolons in extended prose.
- Prose over bullet points. Sentence-paragraph-section-document coherence.
- No exaggeration. Measured academic tone.
- Econ top journal style. Concise and direct over beautiful.
- Results: "We find that ...", "Table N reports ...".
- Citations at end of sentence.

## Environment

- OS: Mac (local dev) + Linux (server)
- Python env manager: uv
- R env manager: renv
- Always use type hints in Python.
