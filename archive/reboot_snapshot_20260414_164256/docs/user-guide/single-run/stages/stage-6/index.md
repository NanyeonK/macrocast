# Stage 6 — Statistical Testing

Stage 6 defines formal model-comparison and predictive-accuracy tests.

## What Stage 6 owns

- equal predictive ability tests
- nested model tests
- instability and conditional predictive ability tests
- multiple-model data-snooping corrections
- dependence correction choices

## Main question

After metrics are computed, which forecast differences are statistically meaningful?

## Separation rule

Metrics and rankings belong to Stage 4.
Statistical testing belongs here.

## Current status

The package taxonomy includes this layer, but the user-facing stage documentation is still ahead of fully expanded runtime wiring.
