# Stage 7 — Variable Importance / Interpretability

Stage 7 explains why a model is behaving the way it does after forecast quality has already been assessed.

## What Stage 7 owns

- model-native importance
- model-agnostic importance
- SHAP-family methods
- grouped and temporal importance
- aggregation and output style for interpretation

## Main question

Once a forecasting path is accepted as meaningful, how do we interpret the driver structure responsibly?

## Ordering rule

Interpretation should follow evaluation and statistical testing.
The package should answer:
1. did it work?
2. is the difference meaningful?
3. why is it working?

## Current status

Interpretation is in scope for the package, but the stage-first user guide is still being built out.
