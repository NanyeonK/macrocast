# macrocast.viz

`macrocast.viz` contains the public plotting helpers for forecast comparisons and interpretation outputs.

## Import

```python
from macrocast.viz import plot_rmsfe_heatmap, marginal_effect_plot
```

## Plotting functions

Visualization exports:
- `marginal_effect_plot`
- `variable_importance_plot`
- `cumulative_squared_error_plot`
- `plot_rmsfe_heatmap`
- `plot_horizon_lines`
- `plot_mcs_membership`

Purpose:
- generate publication-oriented figures for forecast performance and interpretation summaries
