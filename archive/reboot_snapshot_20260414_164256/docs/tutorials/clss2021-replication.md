# CLSS 2021 Example

CLSS 2021 is treated as one recipe/path example, not as package-defining runtime logic.

Canonical artifact:
- `recipes/papers/clss2021.yaml`

Recommended use:
1. inspect the recipe
2. compile it through the recipe-native path
3. use it as one example of how a paper study fits into the generic package

The package should support many paths like this, not a CLSS-specific execution mode.
