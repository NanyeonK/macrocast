# CLSS 2021 Replication

`macrocast.replication.clss2021` provides presets for reproducing the information set
experiment in Coulombe, Leroux, Stevanovic, and Surprenant (2021),
"Macroeconomic data transformations matter," *International Journal of Forecasting* 37(4): 1338–1354.

## CLSS2021 Class

```python
from macrocast.replication.clss2021 import CLSS2021

# Get all 15 information set specs with paper defaults
specs = CLSS2021.info_sets()

# Override paper hyperparameters
specs = CLSS2021.info_sets(P_Y=6, K=4, P_MARX=6)
```

### Default Parameters

| Parameter | Value | Description |
|-----------|-------|-------------|
| `P_Y` | 12 | AR lags of target |
| `K` | 8 | Number of PCA factors |
| `P_MARX` | 12 | MARX lag order |

### Information Sets

The 15 information sets from Table 1 of CLSS (2021):

| Name | Description |
|------|-------------|
| `AR` | AR lags of target only |
| `F` | PCA factors only |
| `F-AR` | PCA factors + AR lags |
| `F-MARX` | PCA factors + MARX lags |
| `X` | X-factors (untransformed predictors) |
| `X-AR` | X-factors + AR lags |
| `X-MARX` | X-factors + MARX lags |
| `MAF` | MAF factors |
| `MAF-AR` | MAF factors + AR lags |
| `MAF-MARX` | MAF factors + MARX lags |
| `Hamilton` | Hamilton (2018) filtered factors |
| `Hamilton-AR` | Hamilton factors + AR lags |
| `Hamilton-MARX` | Hamilton factors + MARX lags |
| `D` | First differences (classical preprocessing) |
| `D-AR` | First differences + AR lags |

For the full replication workflow, see the [CLSS 2021 tutorial](../tutorials/clss2021-replication.md).
